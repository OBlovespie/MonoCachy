MonoCachy is a Cachyos/hyprland rice that uses a monochrome/greyscale colour scheme, as of now there is no dynamic theme-switching or wallpaper switcher features as the whole colour scheme is hardcoded into most configs right now and is not variable. 
                                                     
   



Below are examples of the config, as of now there is still parts that have inconsitencies that I will fix when I have time such as the rounded corners in certain places like in the notification menu and other popups, The distances of popups from edges of screen and more. The Wlogout screen needs more configuring and is quite bare right now.
